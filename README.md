# Bank Account Simulator (Java)

Un petit simulateur de banque en Java, réalisé pour pratiquer les concepts d’OOP, les structures de données et l’interaction utilisateur en console.

---

## Fonctionnalités

- Créer un compte bancaire avec numéro, nom et solde initial
- Déposer de l’argent
- Retirer de l’argent (avec contrôle des fonds)
- Transférer de l’argent entre comptes
- Afficher tous les comptes existants avec leurs soldes

---

## Technologies

- Java 25
- Programmation orientée objet (OOP)
- Gestion de listes avec `ArrayList`
- Console / CLI simple pour interaction utilisateur
